# Medical Appointment Booking Website

This repository contains the frontend code for the Medical Appointment Booking website, developed as part of the IBM Frontend Development Capstone Project.

## Project Description

The Medical Appointment Booking website allows patients to:
- Browse available doctors with profile information
- Book medical appointments online
- View appointment details
- Sign up and login to manage their bookings

## Repository Structure

- **Navbar/** - Navigation bar component with HTML and CSS
- **SignUp/** - User registration page
- **Login/** - User authentication page
- **images/** - Doctor profile images and other assets

## Setup Instructions

1. Clone this repository
2. Navigate to the project folder
3. Open the HTML files in your browser to view the pages
